README

To use the typeForm integration Apps you need to have a TypeForm Keychain associated to your account:
{
	typeform: {
		apiField: "key",
		apiKey: "<your TypeForm API key>"
	}
}
In l5 of the TypeForm Integration Form, change the URL to point to your typeform survey URL
In TF Integration Result App configuration, change the url to point to your typeform survey URL
