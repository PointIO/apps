To setup the DocuSign Contract App:
- Import all the json Templates from the templates folders
- Grab the fact ids of those templates, and update the RELATEDS Maps in the App configuration
- Update the storage link in the App configuration to one of your APIdoc Folder Link id

Troubleshooting:
1. The Token to Box has expired:
Update the token from the OAuth Apps, or check https://pointio.atlassian.net/browse/DEMO-136
